Public Nextcloud URL (advanced)

Bee Flow Cloud calls your Nextcloud back at this URL to verify ownership during first-time setup and for runtime user-sync. Leave blank if your Nextcloud is already publicly reachable at … — that's the common case. Set this only if your Nextcloud is behind NAT or only reachable via a tunnel / reverse-proxy you control.

HTTPS recommended. Example: https://abc123.trycloudflare.com for a temporary cloudflared tunnel, or your reverse-proxy URL.

⚠️ Locked by BEEFLOW_NC_PUBLIC_URL env — unset it via occ app_api:app:setenv bee_flow BEEFLOW_NC_PUBLIC_URL '' to use this field.

Rotate tenant key

    

        Drops the cached tenant key and asks the SaaS to mint a fresh one bound to the
        same organisation. Use this if the current key is suspected leaked, or when
        rolling credentials as part of regular hygiene. The connector keeps the
        previous key as a fallback if the rotation fails, so a momentary SaaS
        outage can't strand the install.
    

    

        
        
    

    






    
Tenant key diagnostic

    

        Run this when sign-in fails with “Connector token rejected: no matching tenant key”
        or you keep seeing 502 Bad Gateway on /api/*. The connector asks
        the SaaS what it has stored for this Nextcloud instance and compares fingerprints — no
        secret material is exposed.
